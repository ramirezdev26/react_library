# Book Dashboard

Aplicación de búsqueda de libros construida con React + Vite.

## Requisitos

- Node.js instalado

## Instalación y ejecución

```bash
npm install
npm run dev
```

Abre el navegador en `http://localhost:5173`.

## Uso

- Escribe el nombre de un libro en el buscador y presiona **Buscar** o **Enter**.
- Usa el botón superior derecho para alternar entre modo claro y oscuro.
